#### 题目描述

给出N个等腰三角形,每个三角由x,y,m三个参数来确定,这三个参数确定三角形的三个点分别为 (x;y), (x+m;y) and (x; y+m)

---

#### 输入格式

第一行给出数字N,N<=2000 下面N行给出Xi,Yi,Mi -10^7<=xi<=10^7 -10^7<= Xi,yi<= 10^7 0 < mi< 1000

---

#### 输出格式

输出三角形覆盖的总面积,答案保留一位小数

---

#### 样例输入
```
5
-5 -3 6
-1 -2 3
0 0 2
-2 2 1
-4 -1 2
```

---

#### 样例输出
```
24.5
```

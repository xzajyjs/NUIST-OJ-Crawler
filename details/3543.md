#### 题目描述

给N个点，问存在多少个两边与坐标轴平行的正方形，四个顶点属于这N个点中的4个。

---

#### 输入格式

第一行一个整数N。  
接下来N行每行两个数x\_i，y\_i表示坐标。

---

#### 输出格式

一行一个整数表示答案。

---

#### 样例输入
```
6
0 0
0 1
1 0
1 1
3 0
3 1


```

---

#### 样例输出
```
1


```

---

#### 提示

【数据范围】  
N<=10^5，|x\_i|,|y\_i|<=10^6

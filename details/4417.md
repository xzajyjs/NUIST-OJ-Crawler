#### 题目描述

现有一个n行m列的棋盘，一只马欲从棋盘的左上角跳到右下角。每一步它向右跳奇数列，且跳到本行或相邻行。跳越期间，马不能离开棋盘。例如，当n = 3, m = 10时，下图是一种可行的跳法。

 ![](../file/4417_0.jpg)

试求跳法种数mod 30011。

---

#### 输入格式

仅有一行，包含两个正整数n, m，表示棋盘的规模。

---

#### 输出格式

仅有一行，包含一个整数，即跳法种数mod 30011。

---

#### 样例输入
```
3 5
```

---

#### 样例输出
```
10

```

---

#### 提示

对于100%的数据，1 ≤ n ≤ 50，2 ≤ m ≤ 10^9

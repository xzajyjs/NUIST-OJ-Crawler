#### 题目描述
给定 $3$ 个数，$a, b$ 和 $c$ ($3 < a, b, c  \leq  10000$)，求出 $a, b$ 和 $c$ 的最大公约数。

#### 输入描述
多组输入，输入的第一行为一个正整数 $n$，表示接下来有 $n$ 组数据，每一行为 3 个正整数 $a, b, c$ ($3 < a,b,c  \leq  10000$)。

#### 输出描述
对于每一组数据，求出 $a, b$ 和 $c$ 的最大公约数。

#### 样例输入

```
2
2 3 4
3 6 9
```

#### 样例输出

```
1
3
```

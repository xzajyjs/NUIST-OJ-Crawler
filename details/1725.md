#### 题目描述

小L特别喜欢数字5，尤其是那一串只由 5 构成的数 5,55,555,5555......

现在小L想知道最小的正整数N，使得 55...5 (N个5) mod m=K. 

#### 输入描述

第一行一个正整数 T，表示组数，接下来 T 行每行两个正整数 K, m.

对于 100% 的数据, 10 < m < 2e9, 0 < K < m，1 <= T <= 5000，保证 m 是个质数.

#### 输出描述

如果存在 N，输出符合条件的最小正整数 N，不存在则输出 "No!";

#### 样例输入
```
2
11 17
2 23
```
#### 样例输出
```
3
No!
```

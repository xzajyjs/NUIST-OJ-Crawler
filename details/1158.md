#### 题目描述
> 试求 n &times; m 的方格图形中，从点 (0, 0) 到点 (n, m) 的最短路径数目。

#### 输入描述
> 有多组测试数据。输入的第一行为一个正整数 N($1 \leq N \leq 10$)，表示接下来有 N 组测试数据。

> 在接下来的每行输入中有两个整数 n 和 m ($1 \leq n,m \leq 10$)，表示该方格图形的大小。

#### 输出描述
> 对于每组测试数据，输出一行答案，即符合要求的最短路径数。

#### 样例输入
```
2
5 4
1 1
```
#### 样例输出
```
126
2
```

#### 输入格式

所有数据都是非负整数。第一行为 W、H （不超过10000），表示大矩形的长和宽。第二行为N、K（1 ≤ K ≤ N ≤ 2000）。接下来 N行，每行四个整数X1、Y1、X2、Y2，表示小矩形对角两顶点的坐标，满足 0 ≤ X1 < X2 ≤ W 0 ≤ Y1 < Y2 ≤ H

---

#### 输出格式

只有一行：R的面积。

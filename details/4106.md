#### 题目描述

有一个数列price\_k = p \* (sin(a \* k + b) + cos(c \* k + d) + 2).

求最大的(price\_i - price\_j) (1 <= i <= j <= n)

---

#### 输入格式

多组数据.每组数据六个整数p, a, b, c, d, n.

---

#### 输出格式

每组数据输出一行表示答案,保留六位小数.

---

#### 样例输入
```
392 456 222 152 222 348222

```

---

#### 样例输出
```
1556.786855

```

---

#### 提示

1 <= p, a, b, c, d <= 1000, 1 <= n <= 1000000.

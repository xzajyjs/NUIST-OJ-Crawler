#### 输入格式

第1行输入N，之后N行每行描述一条水管，前两个英文字母表示水管的两端（大小写字母是不一样的），后一个整数表示水管的流量，流量不会超过1000.

---

#### 输出格式

一个整数，表示总流量．

---

#### 样例输入
```
5
A B 3
B C 3
C D 5
D Z 4
B Z 6
```

---

#### 样例输出
```
3
```

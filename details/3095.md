#### 题目描述

给定一个长度为n的整数序列x\[i\]，确定一个二元组(b, k)使得S=Σ(k\*i+b- x\[i\])^2（i∈\[0,n-1\]）最小

---

#### 输入格式

第一行一个整数n;

第二行接下来n个整数，为{x\[i\]}。

---

#### 输出格式

 包含一行两个实数为二元组(b,k)

---

#### 样例输入
```
4
0 0 0 0 

 

```

---

#### 样例输出
```
    0 0

```

---

#### 提示

    对于100%的数据n<=1000000,每个x\[i\]的绝对值不超过10^8  
   
   你的S不超过标准答案的100.01%时，被认为是可以的。

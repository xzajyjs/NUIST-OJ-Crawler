#### 题目描述

给定平面上(N <= 1500)个草莓的坐标，都是整数，没有两个草莓重合。现在郑爽要选出4个草莓，构成一个矩形，这样才能和翰哥哥一起玩“爱的华尔兹“。为了更浪漫，爽妹还要这个矩形面积最大。

---

#### 输入格式

第一行为一个整数N 接下来N行，每行两个数代表坐标，绝对值小于等于10^8

---

#### 输出格式

一个整数代表爽妹想要的最大的矩形面积。

---

#### 样例输入
```
8
-2 3
-2 -1
0 3
0 -1
1 -1
2 1
-3 1
-2 1

```

---

#### 样例输出
```
10
```

#### 题目描述

给出平面上N个点,N<=10^5.请求出一个半径最小的圆覆盖住所有的点

---

#### 输入格式

第一行给出数字N，现在N行，每行两个实数x,y表示其坐标.

---

#### 输出格式

输出最小半径，输出保留三位小数.

---

#### 样例输入
```
4
1 0
0 1
0 -1
-1 0


```

---

#### 样例输出
```
1.000
```

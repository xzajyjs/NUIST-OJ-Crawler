#### 题目描述
给定 $2$ 个数，$a$ 和 $b$ ($1 < a, b  \leq  10000$)，求出 $a$ 和 $b$ 的最小公倍数。

#### 输入描述
多组输入，输入的第一行为一个正整数 $n$，表示接下来有 $n$ 组数据，每一行为两个正整数 $a, b (1 < a, b  \leq  10000)$。

#### 输出描述
对于每一组数据，求出 $a$ 和 $b$ 的最小公倍数。

#### 样例输入

```
2
2 4
3 5
```

#### 样例输出

```
4
15
```

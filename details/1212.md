#### 题目描述
> 给定整数 a, b, n 要求计算 (a ^ b) mod n

#### 输入描述
> 多组数据，每组数据一行，为三个用空格隔开的整数 a, b, n

> 1 <= a <= 40, 0 <= b <= 3, 1 <= n <= 500

#### 输出描述
> 每组数据输出一行，为所求值。

#### 样例输入
> 2 3 5

> 2 2 4

#### 样例输出
> 3

> 0

#### 题目描述

小L 信仰的数字 x 满足两个特征：

1. x 是一个质数
2. $x = a^2 + b^2$，其中 a 和 b 是任意正整数

小L 想知道在区间 [L, R] 中有多少 Ta 信仰的数字

#### 输入描述

一行，两个正整数 L, R.

对于 100% 的数据, 满足 1 <= L <= R <= 300000000.

#### 输出描述

仅一行，符合题目要求的数字个数

#### 样例1

输入
```
1 5
```

输出
```
2
```

### 样例2

输入
```
6 66
```

输出
```
7
```

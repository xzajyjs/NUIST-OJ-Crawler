#### 题目描述

一个二维向量(x,y)的权定义为x2+y2。已知一个由n个二维向量组成的集合，求该集合的一个子集，使该子集中的向量和的权尽可能大。

---

#### 输入格式

第1行一个数n，表示n个向量。 下面n行，每行2个实数，表示n个向量。

---

#### 输出格式

1个实数，即向量和最大的权。(精确到小数点后3位)

---

#### 样例输入
```
3
1 1
1 0
0 -1

```

---

#### 样例输出
```
5.000


```

---

#### 提示

n<=100000

#### 题目描述

一次考试共有n个人参加，第i个人说：“有ai个人分数比我高，bi个人分数比我低。”问最少有几个人没有说真话(可能有相同的分数)

---

#### 输入格式

第一行一个整数n，接下来n行每行两个整数，第i+1行的两个整数分别代表ai、bi

---

#### 输出格式

一个整数，表示最少有几个人说谎

---

#### 样例输入
```
3

2 0

0 2

2 2

 


```

---

#### 样例输出
```
1


```

---

#### 提示

  
  
100%的数据满足： 1≤n≤100000   0≤ai、bi≤n
